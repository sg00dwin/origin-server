Get Started with OpenShift Online

A free account on OpenShift lets you create up to three applications for free. Typical OpenShift applications include a web programming framework like JBoss Application Server, Ruby on Rails, or PHP, one or more databases, and additional capabilities if desired. OpenShift provides these capabilities through cartridges that automatically manage themselves. You can also add database admin cartridges like RockMongo and phpMyAdmin, if you wish.

  • IRC Chat for instant discussions and help
  • Maling lists to ask questions or suggest new ideas
  • Community forums are watched round the clock by an army of developers, evangelists, and OpenShift devotees.
  • Knowledge Base for known headaches and FAQ for more general questions.
  • Community Search to find answers on a specific topic.
